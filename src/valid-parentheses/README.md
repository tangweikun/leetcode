# 17.Valid Parentheses

## Description

Given a string containing just the characters '(', ')', '{' , '}', '[', ']', determine if the input string is valid.
The brackets must close in the correct order, "()" and "()[]{}" are all valid but "(]" and "([)]" are not.

## Example

```javascript
Input: {[()]}
Output: true
```
