# 189.Find Bottom Left Tree Value

## Description

Given a binary tree, find the leftmost value in the last row of the tree.

## Example1

```javascript
Input:

    2
   / \
  1   3

Output:
1
```

## Example

```javascript
Input:

        1
       / \
      2   3
     /   / \
    4   5   6
       /
      7

Output:
7
```
