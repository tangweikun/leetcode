# 6.ListInsert

## Description

在线性表 list 的指定位置 position 上插入指定值 elem，如果 position<0 或 position>list.length 就返回原 list

> Similar with `splice(position, 0, elem)`

> Time complexity O(n)

> Linear table sequential storage structure

## Usage

```javascript
listInsert(arr, position, elem)
```
